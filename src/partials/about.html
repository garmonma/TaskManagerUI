About Holistic Nutrition Calculator

Last updated - Monday Sept 5, 2;17 AM


Holistic Nutrition Application is just a test application so I can learn the technologies needed to implement my dream application. However, while I am building this application I will implement the best programming practices and make this application the best that it can be. Scouts honour.

Technologies

  • Mongo DB
  • Angular JS
  • Spring Boot - MVC
  • HTML5 Boilerplate


The application is way for users to track their calories, first and foremost, but also all of the other nutritional values that appear on a typical nutrition label. I would also like to track more of the minerals and macro nutritients, especially those found in fruits ad veggies. The calculator will be the main feature, making it available to everyone, even users without an account. The other features will be accessible to the users that register for a free account. These features include, but not limited, the ability to save caculations, create meals and a weekly meal planner, and to see nutritional value reports of eating habits.
